Battle.net OAuth token
===

Asynd-std compatible fork of [shempignon/battle-net-oauth](https://github.com/shempignon/battle-net-oauth) using Surf instead of Reqwest

Installation
---

```
battle-net-oauth-async-std = "0.1"
```
